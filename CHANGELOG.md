# 更新日志

## v1.0.0 · 2026-06-24

首个公开版本。一个轻量、通用的多租户 OpenAI 兼容 API 转发网关。

- **转发网关**：`/v1/chat/completions`（含流式 SSE 透传）+ `/v1/models`，OpenAI 兼容；Bearer Key 鉴权、模型路由、上游错误透传。任何 OpenAI 兼容客户端都能直接用。
- **管理后台**（苹果风 UI）：管理员首启设密码 / 登录；厂家管理（国内厂家预设一键填）；模型路由；老师账号增删改 + 单建 + CSV 批量导入（姓名,手机号,备注）+ 重置 Key / 重置登录密码；调用统计（每位老师次数、按模型汇总、token 尽力统计）+ 调用日志。
- **老师端自助门户**（`/portal`）：老师用**手机号 + 默认密码 openkeyhub** 登录，**首登强制改密**后自助查看自己的 Base URL + API Key + 可用模型，一键复制。管理员无需单独/公开分发密钥；忘记密码由管理员一键重置回默认。
- **静态加密**：厂家真实 Key、老师 API Key 在库内**加密存储**（HMAC 认证加密，零外部依赖）；管理后台只显示掩码，明文仅老师本人登录可见。
- **账号限额/限速**：可选，默认不限制。
- 本地部署（`start.sh` / `python -m backend.main`），建议只在校内局域网。
